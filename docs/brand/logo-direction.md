# Logo Direction

Working notes for Aredir Labs logo and mark development.

## Objectives

- Readable at small sizes (favicon, nav)
- Works on light and dark backgrounds
- Scales from app icon to hero lockup

## Exploration criteria

- Geometric or wordmark approaches that suggest precision and forward motion
- Avoid overly literal sports clichés unless intentionally brand-defining
- Limit palette in the mark itself; rely on site theme for extended color

## Deliverables (when design is complete)

- [ ] Primary logo (SVG)
- [ ] Mark / icon only (SVG)
- [ ] Favicon set (ICO + PNG sizes)
- [ ] Social share template (1200×630)
- [ ] Usage rules (clear space, minimum size, don'ts)
